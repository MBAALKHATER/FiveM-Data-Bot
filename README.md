# FiveM | Data Bot

#Instalacja

Wymagania:
  NodeJS

Linux
  1. Stwórz folder i wypakuj pliki bota.
  2. Uzupełnij token bota w pliku config.json.
  3. Podmień linki w pliku index.js
    - http://<ip>:<port>/info.json
    - http://<ip>:<port>/players.json
  4. Zainstaluj wymagane pakiety
    - npm install
  5. Uruchom bota
    - node index.js
